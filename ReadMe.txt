A simple Ping pong game. There are two platforms who hits the ball and if the ball touch top or bottom 
wall the score of one of the players increase.

classBall.h - describes ball
classMap.h - describes map
classPlatform - describes platforms 
gameRules.h  - rules of the game
gameMovement.h - descries movement of the game
gameSetUp.h - set up
MapFunctions.h - describes map functions
platformFunctions.h - describes platform functions
